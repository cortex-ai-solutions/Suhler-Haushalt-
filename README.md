# Suhler-Haushalt-
Dieses Projekt bündelt die Seiten des Suhler Haushaltsplans 2025 über eine intelligente, inhaltsbasierte Python-ETL-Pipeline fehlerfrei in einer mehrjährigen SQLite-Datenbank. Das Ziel ist ein statisches, clientseitiges HTML-Dashboard für GitHub Pages, das Stadträten Handlungsspielräume und SGB-Ermessen interaktiv visuell aufbereitet
